🏭 Esterification of Acetic Acid with Methanol – Plant Design
📌 Overview

This project focuses on the process design of a methyl acetate production plant using the esterification of acetic acid with methanol.
It includes process flow development, reactor selection, separation strategy, and simulation.

🔬 Process Description

The plant is based on a liquid-phase esterification reaction:

CH3COOH + CH3OH ⇌ CH3COOCH3 + H2O

The process operates continuously and involves reaction, separation, and recycle of unreacted components.

⚙️ Process Flow
Feed preparation (mixing, pumping, heating)
Reaction in a Plug Flow Reactor (PFR)
Separation using distillation columns
Recycle of unreacted methanol and acetic acid
🧪 What This Project Covers
Process flow diagram development
Reactor selection and design reasoning
Material balance calculations
Separation system design
Process simulation and validation
🛠️ Tools & Technologies
Aspen Plus (RPlug, RADFRAC)
Chemical Engineering design principles
Material and mass balance calculations
Thermodynamic modeling (NRTL)
🎯 Objective

To design a conceptual chemical process plant and understand the integration of reaction engineering, separation processes, and simulation tools.
